## **Projeto Faculdade A3 — Página sobre Meredith Grey**

Este repositório contém um projeto desenvolvido como trabalho da faculdade, dividido em três fases, com o objetivo de aplicar e evoluir conhecimentos em desenvolvimento web ao longo do tempo.

O tema escolhido para o projeto foi a personagem Meredith Grey, da série Grey’s Anatomy, abordando sua história, características e importância dentro da série.

📚 *Sobre o Projeto*

O projeto consistiu na criação de uma página web temática, que foi sendo aprimorada a cada fase do trabalho.
Em cada etapa, novos recursos, melhorias visuais e funcionais foram adicionados, demonstrando a evolução do aprendizado ao longo da disciplina.

🎯 *Objetivo*

Praticar conceitos de desenvolvimento web

Aplicar melhorias progressivas no layout e no conteúdo

Trabalhar organização de código e estrutura de páginas



🧩 *Estrutura por Fases*

🔹 Fase 1

Criação da estrutura básica da página

Conteúdo inicial sobre a personagem

Uso de HTML para organização do site

🔹 Fase 2

Melhorias no layout

Introdução de estilização com CSS

Organização visual mais clara e agradável

🔹 Fase 3

Aprimoramento geral da página

Inclusão de novos elementos e seções

Melhor organização do código e melhor experiência do usuário

📌 *Observações:*

Este projeto foi desenvolvido com finalidade acadêmica, mas reflete a evolução prática no aprendizado de desenvolvimento web ao longo do curso.
